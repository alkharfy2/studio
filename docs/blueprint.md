# **App Name**: Cveezy

## Core Features:

- User Authentication: Secure user authentication using Firebase, with role-based access control for admins, moderators, designers, and clients.
- Task Management: Create, assign, and track CV design tasks, with detailed task information including client details, services requested, and financial data. Assign different access to different roles.
- Financial Calculations: Automatically calculate moderator earnings based on completed tasks, considering deductions for advertising costs. The tool accurately manage financial data using Firestore to track payments, remaining amounts, and currency.
- File Upload and Storage: Enable users to upload and store CVs, certificates, images, and payment receipts using Supabase Storage, with organized folder structures and secure access.
- Notification System: Real-time notification system using Firebase Cloud Messaging to notify users of new tasks, task updates, and completed tasks. Implement in-app notifications with a notification bell.
- Moderator Dashboard: Dashboard for moderators to view financial performance, open tasks, overdue tasks, and recent tasks.  Show only current relevant information.
- Task Creation UI: Design an intuitive and comprehensive task creation interface for moderators to input client data, services, and task details.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and trustworthiness, relating to business and expertise.
- Background color: Light gray (#F0F2F5), offering a clean, modern backdrop that enhances readability and content visibility.
- Accent color: Soft violet (#9575CD), to provide a subtle contrast that does not overpower the professional aesthetic.
- Body font: 'Inter', a versatile sans-serif, used to guarantee a clean, objective feel across the user interface.
- Headline font: 'Space Grotesk', a distinct sans-serif for titles and headlines that combines approachability and scientific objectivity.
- Use a consistent set of minimalist icons from Material Design to represent various actions and categories.
- Implement a responsive layout that adapts to different screen sizes, ensuring a consistent and user-friendly experience on desktops, tablets, and mobile devices.
